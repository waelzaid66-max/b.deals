## لماذا شعرت أن «نصف الشغل فقط»؟

| ما قاله الوكيل السابق | الحقيقة |
|------------------------|---------|
| «M01–M31 منجز» | **كود** موجود — **API حي FRESH** (2026-07-10) — **جهاز لم يُختبر** |

## خطة العمل على مراحل (بدون تخريب)

### المرحلة A — تثبيت الكود (لا ميزات جديدة)
1. مراجعة diff موجات 4+5  
2. `node artifacts/banco-mobile/tests/lib-hardening.test.mjs`  
3. `node scripts/production-confidence-check.mjs`  
4. commit على `fix/mobile-master-stabilize` **عند طلبك فقط**

### المرحلة B — OPS (أنت أو Replit)
1. تدوير الأسرار المكشوفة → `.secrets/local.env` (انظر `scripts/local.env.example`)  
2. Redeploy API من الفرع  
3. `pnpm run ops:post-redeploy` حتى **FRESH**  
4. `pnpm run ops:wave-b` → EAS preview

### المرحلة C — إصلاحات منتج متبقية (بعد B)
1. **هاتف متعدد على البروفايل:** إما مكوّن مشترك + حقل `user.default_contact_phones` في API، أو توضيح UX أن الإعداد في **إنشاء إعلان** فقط  
2. **DEVICE-QA:** تنفيذ `DEVICE-QA-SECTION-COMPANIES.md` صفاً بصف  
3. **ماسنجر:** ضبط حسب لقطات جهازك — لا تخمين من الوكيل

### المرحلة D — لا تفعل
- حذف مسارات business/import/rent «للتنظيف»  
- ادعاء «كل شيء أخضر» من CI فقط
