**🛑 CRITICAL COMPLIANCE ADDITIONS:**
1. **Self-Service Account Deletion:** Google *strictly* requires that if an app allows account creation, it MUST allow account deletion *within* the app.
2. **Explicit Permission Justification:** Every permission request (Camera/Gallery) must be accompanied by a "Permission Rationale" screen before the OS system prompt appears.

## 1. THE MANDATORY DELETE PIPELINE (`api/user/delete-account`)

* **The Flow:** User clicks "Delete Account" -> Trigger Backend API -> API flags user as `anonymized` -> All `leads` from this user are wiped -> Clerk `deleteUser` is triggered.
