🧠 السياق (مهم جدًا)

أنت بالفعل قمت ببناء:

* هياكل البيانات (Data Structures)
* منطق الإعلانات (Listings Logic)
* نظام التجار (Dealer Systems)
* خطط الدفع والمونيتايزيشن (Billing Plans)

لكن المشكلة الحالية:

❌ كل ده غير ظاهر داخل واجهة الموبايل
❌ المستخدم لا يرى أي قيمة حقيقية على الشاشة

وهذا يعتبر فشل حالي في التنفيذ

🎯 وضع التنفيذ الجديد (إجباري)

نحن الآن لا نعمل في مرحلة تخطيط
ولا في مرحلة بناء منفصل

نحن نعمل في:

→ مرحلة الربط + الإكمال + الإظهار داخل الموبايل

أي Feature موجود لازم:

✔ يظهر داخل التطبيق
✔ يكون قابل للضغط
✔ يكون قابل للاستخدام

أي شيء غير ظاهر = غير موجود

🚫 قواعد صارمة

* ممنوع حذف الـ Task Queues
* ممنوع إيقاف الباك إند
* ممنوع بناء Features منفصلة غير مربوطة

بدلاً من ذلك:

→ اربط كل شيء بالشاشات داخل الموبايل



📱 الأولوية القصوى: إظهار التطبيق

ابدأ فورًا بـ:

1. الصفحة الرئيسية (Home Feed)

* عرض بيانات حقيقية (سيارات / عقارات / صناعي)
* استخدام SmartAssetCard ويحتوي على:
    → صورة أو فيديو
    → السعر بشكل واضح
    → التقسيط (إن وجد)


2. صفحة التفاصيل (Listing Detail)

عند الضغط على أي إعلان:

يجب فتح صفحة تحتوي على:

* معرض صور كامل
* السعر
* خيارات الدفع (إسلامي / بنكي / مورد)
* أزرار:
    → اتصال
    → واتساب
    → طلب عرض

3. التنقل داخل التطبيق (Navigation)

تأكد أن هذه الأقسام تعمل بالكامل:

* Home
* Search
* Create Listing
* Profile

وكل قسم متصل فعليًا

🧩 ربط الأنظمة الموجودة (مهم جدًا)

لا تبني جديد

خذ الموجود واربطه:

* بيانات التجار → تظهر في الإعلانات
* خطط الدفع → تظهر في التفاصيل
* نظام Leads → يعمل عند الضغط على Contact

❗ لا تنشئ أنظمة جديدة
❗ فقط ربط + إكمال

🎬 شاشة البداية (Cinematic Splash)

إنشاء تجربة افتتاحية احترافية:

* خلفية سوداء كاملة (#000000)
* خطوط متحركة تمثل:
    → سيارات
    → عقارات
    → معدات صناعية
* حركة ناعمة وبطيئة (Premium feel)
* لوجو BANCO في المنتصف:
    → يظهر تدريجيًا (fade)
    → تكبير بسيط (scale)
* المدة:
    → من 1.5 إلى 2 ثانية فقط

ثم الانتقال السلس إلى الصفحة الرئيسية

🧪 الاختبار النهائي (إجباري)

لن يعتبر العمل منتهي إلا إذا:

المستخدم يستطيع:

→ فتح التطبيق
→ رؤية الإعلانات
→ الضغط على إعلان
→ مشاهدة التفاصيل
→ الضغط على تواصل

بدون أي خطأ أو تعقيد

🚀 الهدف النهائي

التطبيق يجب أن:

✔ يعرض قيمة حقيقية
✔ يعمل بالكامل من الموبايل
✔ يخدم المستخدم مباشرة
✔ يكون جاهز للنشر

📌 تعليمات نهائية

* لا تقم بإرسال تقارير مرحلية
* لا تتوقف عند تنفيذ جزئي
* لا تبدأ أي شيء جديد

ابدأ بـ:

→ ربط Home Feed + Listing Detail فورًا
→ ثم أكمل باقي الشاشات بنفس المنطق

وأكمل حتى يصبح التطبيق جاهز بالكامل
